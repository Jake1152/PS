전화번호 목록 26NOV20
 
전화번호부에 적힌 전화번호 중 한 번호가 다른 번호의 접두어인 경우가 있는지 확인
전화번호가 다음과 같을 경우, 구조대 전화번호는 영석이의 전화번호의 접두사
구조대 : 119
박준영 : 97 674 223
지영석 : 11 9552 4421
전화번호를 담은 배열 phone_book이 매개변수로 주어 질때
어떤 번호가 다른 번호의 접두어 인 경우가 있으면 false
그렇지 않으면 True

제한 사항
phone_book의 길이는 1 이상 1,000,000 이하입니다.
각 전화번호의 길이는 1 이상 20 이하입니다.

# 문제이해
어떤 전화번호가 다른 전화번호의 시작이 될 때 False
그런 경우가 하나도 없으면 True


# 문제해결방법
해시랑 무슨 상관인가?
오름차순으로 정렬 한 뒤 짧은 번호부터 이중 반복문으로 다른 번호의 접두어인지 확인 
접두어이면 return False로 바로 종료

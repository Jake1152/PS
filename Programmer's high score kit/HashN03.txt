# 문제이해
위장 소품을 1개씩은 사용해서 중복되지 않게 매번 다르게 입으려고 한다.
그럴 경우 가짓수를 return

바지만 입을 수는 있어도 얼굴에 안경쓰고 선글라스 쓰고 
한 부분에 2가지 이상의 소품을 쓸 수 없다.
한 부분에는 1개만 쓸 수 있고
신체 전부에 적어도 1개 이상의 소품을 써야한다.

# 문제해결방법
- 반복문 밖에 빈 해시,sum변수 생성
- 주어진 2차원 문자열 배열을 반복문 돌린다.
- 배열의 마지막에 소품 이름이 있다.
  소품이름을 해시의 키값으로 이용하며
  해시 값으로는 배열을 사용해서 추가한다 
 
- 소품별로 갯수+1씩해서 (안 입는 경우 포함)
  모든 소품+1의 갯수를 곱해준다
  결과값으로는 전체 갯수에서 -1을 한다(적어도 1개는 입으니까 아예 안 입는 경우 제외)

X 
- 해시 길이가 1이면 해시 키에 해당하는 값(배열)의 원소 갯수를 return
- 해시 길이가 1보다 크다면
  sum에 아래 값들을 더해준다 뒤 return한다.
  - 해시 항목별 배열의 원소 갯수를 전부 더한다.(1개의 소품을 쓰는 경우 감안)
  - 해시 항목별 갯수들을 모두 곱한 값
